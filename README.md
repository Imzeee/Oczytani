# Oczytani

## Run on localhost

1. Clone repository 
```
git clone https://github.com/yuri3262/Oczytani.git
cd Oczytani
```
2. Install dependencies
```
sudo ./install.sh
```
3. Run chrome webdriver and application
```
sudo ./run.sh
```
4. Go to localhost:8080 to test application
5. (Optional) Run tests
```
sudo ./test.sh
```
